# jquery.activeChanger

## options

**controler**  
show controler  
```
default: true
options: boolean
```

**pager**  
show pager  
```
default: true
options: boolean
```

**delay**  
before start delay  
```
default: 0
options: integer
```

**duration**  
active pause speed (ms)  
```
default: 4000
options: integer
```

**prevText**  
prev control text  
```
default: prev
options: string
```

**nextText**  
next control text  
```
default: next
options: string
```
